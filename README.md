# pyACH [![Semver](http://img.shields.io/SemVer/0.1.3.png)]()

## Requirement
```
python 2.7
npm 3.10.10
nodejs 6.9.4
```

## Installment
```
$ virtualenv env
$ pip install -r requirements.txt
$ npm install
$ sudo npm install -g pm2
```
## Start server
```
$ source ./env/bin/activate
$ pm2 start run_server.py
```
